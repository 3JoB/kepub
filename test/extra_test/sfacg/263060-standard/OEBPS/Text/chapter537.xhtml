第三十七章 轻敌的勇者随从与冒险的玛格丽特

「伊丝！不要拉着我！下面可是你的同伴！」

当红使用底牌『碎龙』时，一直在高空俯瞰战场的琳便不再旁观，伴随背后衣料发出的「呲拉」声，拍打着巨大龙翼，动身前往下方玛丽一行所在的位置，『碎龙』的一击有多大威力，身为红同伴的琳简直再清楚不过，那是能够与成年龙正面抗衡的，红竭尽全力的一击。

然而，伊丝却在琳动身的那一刻伸手搭在琳肩膀上，阻止琳的前往。

「等等，琳，你忘记触发式魔法道具了吗。」

「没有任何触发式魔法道具能抵挡红的全力一击，『碎龙』的威力已经到四阶魔法水平了！」

扭动肩膀甩开伊丝，琳不再与伊丝对话，直接以突破六十级以后的最快速度，赶往红即将砸中玛丽的地方，但即便是超越英雄领域的速度，在如此距离下也不可能赶上，琳只能眼睁睁看着同伴的攻击与玛丽碰撞在一起。

来不及了。

简直胡闹。

内心咒骂的同时，琳被迫停止救援，在毫无准备的情况下涉足『碎龙』的覆盖范围，即便是自己也会遭到重创，而同一时间，紫也从湖泊的方向以『飞行』魔法赶来，手里还提着嘴巴呈へ状，一副不高兴表情的蒂芙尼，从两人浑身是水，衣服和头发被淋湿个透的样子看，显然也经历了一番战斗。

「琳，这边怎么了？」

意识到情况不妙，紫的眉毛拧在一起。

「红用了『碎龙』，她被埋伏了，这是卡米拉她们商量好对抗你们的陷阱。」

「『碎龙』？怎么可能！虽然我想到她们是想利用这只史莱姆争取时间，但这才过去了不到两分钟，她们不可能把红逼到使用『碎龙』。」

「具体发生了什么之后再告诉你。」

只有从高空俯瞰战场的伊丝、拉克西丝与琳三人才知道具体发生了什么。

这是只存在于切磋中的策略。

当红与紫决定分开的时候，卡米拉一行的计划便顺利展开。首先是蒂芙尼为卡米拉施加『强效之祝福』，增加卡米拉下一个魔法，也就是『冰枪术』的威力，在这之后，蒂芙尼便被玛丽运送到空旷的、极易被『探查之眼』发现的湖泊附近，以此吸引紫的注意力，将紫带往远离战场的地方，争取尽可能多的时间。

至于拖延时间的方法，原本在卡米拉的设想中，战斗力近乎为零的蒂芙尼，能让紫多花些时间在来回奔袭上就已经十分不错，但出人意料的是，蒂芙尼借助身为史莱姆的特点，选择将一部分身体融入进湖泊，就像在安东尼奥城时，伊丝见过的那些相互吞噬、融合的史莱姆一样，蒂芙尼将大量湖水吸收进身体，进而使身体巨大化，役使湖水攻击紫。

虽然是没有什么威力的进攻手段，就算最基础的一阶魔法『魔力盾』都能抵挡喷射的水流，但由于蒂芙尼使用了具有十足迷惑性，以至于会被打上腹黑标签的计划，激怒了紫，这才拖延了接近一分钟时间。

——首先，当紫第一次使用『探查之眼』观察森林的时候，由于探查之眼会在森林上空形成巨大的白色眼球，目标十分明显，那时候的众人便躲藏在树冠下，仅仅让被卡米拉施加了『浮游之术』的依蒂丝升上高空，借此吸引二人注意力，可众人没想到的是，她们原以为紫和红会在依蒂丝升空的时候就发现依蒂丝，进而分头行动——一个人解决依蒂丝，另一个去追赶其他还没有出现的家伙，一刻钟的限制注定了她们必须要分头行动，可那时『探查之眼』却并没有切到依蒂丝上升的画面（探查之眼的观测范围固然很广，但显示出来的画面大小却会因使用者的选择而异），所以依蒂丝才不得不使用投枪，以此吸引二人注意。

随后，当紫与红分开时，早已被玛丽送到湖泊边的蒂芙尼，便开始吸纳湖泊里的水分，等到其将自身与湖泊融为一体，变的足够巨大的时候，蒂芙尼便把身体的一部分拟态成自己的模样，令其在湖边做出逃跑姿态。

于是，通过『探查之眼』观测到这一幕的紫，十分自然的将拟态认为是正在四散逃跑的蒂芙尼，为了抓住蒂芙尼，紫降低了飞行高度，而这时，蒂芙尼给了紫一个惊喜。

站在高空视角看，小半个平静的湖面就像活过来了一样，突然张开巨口，涌起波涛，由下而上的将掠过湖面的紫一口吞下，与此同时，蒂芙尼把早已准备好的，夹杂了鱼虾、贝壳与海草等一些小型动植物的水流全部灌向紫，虽然及时展开防御魔法的紫没有受到任何伤害，『避水的加护』也防止了呛水可能，但被一个弱小的史莱姆埋伏，这玷污了紫身为魔法咏唱者的高贵，再加上全身湿透、衣服上也满是鱼腥和淤泥的臭味——当紫从湖底出来时，衣服里甚至还有两三只鱼苗在乱窜——怒不可遏的紫便无差别的对着湖水使用魔法轰炸，哪怕蒂芙尼第一时间用上触发式魔法道具，也仅仅支撑了不到半分钟，后来的时间里，蒂芙尼很快便被打回原形。

然而也就是这个原因，导致紫忽视使用『探查之眼』，观察红的方向，失去了最佳的救援时间。当然，倘若这不是切磋，而是生死决斗，紫无论如何也不可能与红分开，更不可能不去看红的情况，但正因为是切磋，再加上对手过于弱小，才令紫松懈下来，进而演变成如此局面。

值得一提的是，这点对卡米拉一行来说也同样如此，如果这不是切磋，而是生死决斗，卡米拉无论如何都不会选择牺牲蒂芙尼争取时间。

等到烟尘散去，众人很快看到了令她们瞠目结舌的一幕。

除了红异常狼狈以外，其他人仅仅只是衣服和头发被风吹乱，全身被一层蛋壳般的淡金色光罩笼罩，只有那些距离碰撞点过于近的人，蛋壳般的结界上才显现裂纹。

降低飞行高度的伊丝与拉克西丝来到琳身边，伊丝一脸无奈的耸肩。

「怎么样，没有危险吧，琳你太担心了，我的——我朋友铸造的魔法道具可以完全放心。」

不过如果换成魔法咏唱者的紫来，或者红的等级突破39，到40级的话，自己就不会像刚才那样轻松了，目前的『壁障手环』最高只能抵御三阶魔法，倘若是四阶魔法，或者附加了『贯破』、『破魔』效果的三阶魔法攻击，『壁障手环』就没有了防御效果。

悬着的心终于放下，琳忍不住抚著胸口，大大的松了口气，与其他人一同解除『飞行』魔法，落到地面。

「呼，还好没人受伤。」

「红，你怎么能用『碎龙』！」

不明白到底发生了什么的紫连忙跑到红身边，一边责怪似的质问，一边关切的为红使用生活魔法，不同于其他人，没有触发式魔法道具保护的红全身沾满灰尘，右拳也因为用力过度，无法遏制的微微颤抖。

「……哈啊，是我输了，不过看紫你的样子，应该还能再战吧。」

「说什么傻话，都这样子了还想着战斗。」

「不行，我必须要说，如果这是没有障碍物的场地，我一个人就能取胜，不过输了就是输了，她们是值得钦佩的对手。」

「你脑袋里全是肌肉吗，快找个地方坐下来换件衣服。」

当紫强行推搡着红的时候，卡米拉一行也来到伊丝身边，因为战胜了远强于自己的红，包括依蒂丝在内，所有人脸上都带着喜悦。

「姐姐大人……」

大概是在等自己夸奖吧。

卡米拉的表情仿佛在这么说。

虽然觉得不太合适——主要是年龄上，卡米拉与小孩子一样的玛丽不同，是成年的美丽少女，但伊丝还是摸了摸卡米拉的头。

「做的不错。」

由于感到另一边的玛丽也一副期待氛围，以至于让伊丝觉得如果玛丽有尾巴的话，肯定会像小狗一样摇来摇去，事实上，小幅晃动的精灵耳已经证实了伊丝的想法，伊丝又夸奖般的摸了摸玛丽的脑袋。

「诶嘿嘿……」

全身都散发出了舒适的气氛。

不过大概是觉得自己的份被玛丽抢走，卡米拉鼓起嘴，喉咙里发出不满的「姆」声，致使玛丽全身一僵。

哈啊，明明已经成年，却还要和玛丽生气，卡米拉还没有长大啊。

不禁有种哭笑不得的感觉，伊丝只能苦笑着伸出两只手，卡米拉与玛丽一边一个，随后，伊丝又把目光放到一旁不时看向自己这边的依蒂丝，好奇的歪了歪头。

「佩佩和蕾菲娜也要吗？」

「啧，把我当小孩子吗？还有你终于开始打蕾菲娜的主意了么。」

瞥见蕾菲娜扇动着的吸血鬼翅膀突然停下，面色扭捏的模样，伊丝连忙摆手。

「不不，我绝对没有那种想法。」

浑身湿透的蒂芙尼也来到伊丝面前，一把抱住伊丝腰部，下巴埋进伊丝微微起伏的胸前，由下向上的，不满的指了指嘴巴。

「饿了。」

「是是，我知道了我知道了，蒂芙尼是这次的功臣，要好好奖励一下，回去以后想吃什么都可以。」

「什么都可以？」

如同小孩子一样歪了歪头，再加上面无表情的样子，蒂芙尼散发出与玛丽不一样的可爱。

「嗯，什么都可以。」

「那，龙肉。」

「呃……这个嘛……」

这家伙还真敢说啊。

虽然旁边就有一头巨龙，但那是和神教签订了契约，与神教有同盟关系的存在，如果把它宰掉的话，琳会把自己吃了吧——只要尾巴说不定没问题？

危险的想法刚浮现出脑海，伊丝便立即将其压下去。

不，还是随便找一头地龙那种程度的亚龙好了，蒂芙尼也没说具体是什么龙的肉。

「吾主，这场切磋就以红和紫的胜利为结束吧，尽管我们之间的配合让红耗尽魔力，但紫还保有完全的战力，如果紫以飞行魔法飞到天空，再以魔法轰击，我们没有办法处理她，从一开始，我们采取的策略就太冒险了，如果蒂芙尼没有争取到如此时间，我们只会被一网打尽。」

拉克西丝走到伊丝身边，低头汇报着看到的一切。

拉克西丝身为司教之天使，在战斗方面的判断上从未出错，而且事实也正如拉克西丝说的那样，紫还保有大半魔力，如果继续战斗下去，魔力耗尽的同伴们肯定不是对手。

将飘到身前的双马尾拂至身后，伊丝思索着点头。

「嗯，就这样吧，本来就太过勉强，我低估了红和紫的实力。」

在自己的设想里，『壁障手环』既然能完全抵挡任何不带『贯破』、『破魔』效果的三阶魔法，那么39级，最高只能使用三阶魔法的红与紫就应该拿『壁障手环』没办法，然而事实是，先不说魔法咏唱者出生的紫，就连以双拳作为武器，破坏力没那么强的红，都可以打碎『壁障手环』的防御结界。

「等等，伊丝，我认为可以按平局处理，紫和红的表现太让人失望了，虽然我猜到她们可能会轻敌大意，但没想到红竟然会落败，如果被教官时期的亚特伍德看见，红至少会被关一周禁闭，现在我身为领队，要让她们知道自己的错误才行。」

如同二人的长辈般，琳一脸严肃的样子，好看的紫绀色眼眸中充斥着不满。

「说成平局就太刻意啦，卡米拉觉得呢？」

「我和拉克西丝的想法一样，虽然我们耗尽了红的魔力，但剩下来的紫我们无论如何也无法战胜，所以应该以我们的失利作为结果，这是无法辩驳的事实。」

露出『你看就是这样』的表情，伊丝朝琳摊手。

皱起眉毛思考片刻后，琳叹了口气。

「好吧，不过回去以后我一定会好好教训她们。话说回来，伊丝这次的触发式魔法道具好像和上次的不太一样？没有攻击性了，之前那个火焰风暴着实吓了我一跳。」

「嗯，我把攻击能力改成了辅助强化佩戴者，这样就能避免误伤出现，」比如蕾菲娜的迟缓攻击，佩佩的身体强化，「只是『壁障手环』的防御性能比我想象中弱了一些，之后得想想该怎么进一步强化才行。」

「伊丝，你刚才的话是认真的吗？」一副下巴快要掉到地上的表情，琳夸张的张大嘴，「连『碎龙』一击都能挡住的防御性能，居然说弱？要是那些大贵族听见你的话，肯定会怀疑自己是不是没见过世面，连赫尔斯王国国王身上的触发式魔法道具，都没有伊丝你们身上的好。」

「欸。」

虽然很想问琳怎么会知道赫尔斯王国国王身上的触发式魔法道具，但总觉得会听到一些恐怖的答案，伊丝十分自觉地转移话题，微笑着歪头，以挑染了蓝色的银发双马尾吸引琳目光。

「正好蒂芙尼肚子饿了，大家久违的在森林里享用料理怎么样？正好天气转凉，我有一种非常适合在这个季节吃的料理哦。」
