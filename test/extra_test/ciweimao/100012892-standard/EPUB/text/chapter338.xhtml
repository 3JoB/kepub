第四十章 智商是硬伤啊少女

更新时间2012-12-19 12：50：46 字数：2091

魅魔的腰部被冈格尼尔狠狠地刺穿了，因为冈格尼尔那个大得比长矛还过分的枪头，这个伤口看上去十分的吓人。

“可…可恶的小丫头…”

一连被一个看上去并没什么实力的人类给连续重伤数次，即使魅魔原本并没有什么恶意，现在也是动了真火。

“恶灵！”

魅魔捂住腰部的伤口，发出了一声号令般的低吼，瞬间，一阵阴风开始在少女的上空汇聚起来。

（操纵怨灵的能力…）

看到这一幕，少女心中想到了魅魔的天赋“操灵”，不禁握紧了手上的长枪，抬头看向天空。

“去！”

随着魅魔的一声令下，少女上空的阴风突然收紧，一条条白茫茫的灵体出现在了上空，不断发出充满怨恨的咆哮，然后向着下面的少女扑去。

（不知道冈格尼尔能不能对付灵体…）

少女咬咬牙，右手握紧冈格尼尔使用了技能“血红之枪－冈格尼尔”，然后用力对着一个已经扑到自己面前的怨灵挥了过去。

然而正如少女所想的，冈格尼尔没有对灵体造成伤害的效果，而是直接从怨灵的身体里穿过，毫无效果。

（我去，明明是北欧神话中的战神之枪，居然没有对付灵体的技能！）

看到这一幕，少女的心顿时凉了半截，一个星幽鬼步闪开，少女的MP顿时掉落到了1300点。

不过那些怨灵并没有关心少女的感受，见少女闪到了一边，天上那些怨灵也转了个方向继续扑向少女。

一时不查，少女被一个怨灵撞了个正着，而一如少女的攻击一般，怨灵撞到少女身上也没有让少女有什么触感，而是从少女的身体里穿了过去。

但是这一穿却让少女感觉到一阵眩晕感，甚至脚下有了些虚浮的感觉，而看到少女的虚弱反应，那些怨灵似乎变得更加兴奋地冲了上来。

（这是怎么回事！？）

咬牙再次使用了星幽鬼步闪开这些怨灵，少女打开属性栏看向了自己的HP，却发现上面显示的数值居然是500/9500。

（我去！就这么一撞就带走我9000的HP！？）

心中大骇的少女脸色大变，看着还剩850的MP，少女心中顿时一片绝望。

就在此时，一声娇喝从上空突然传来。

“博丽巫女封印术式？表壹式？梦想封印散！”

四处乱飞的灵符突然出现在少女身边，如同被狂风吹得四处乱跑的纸屑一般的灵符却精准地击中了每一个怨灵，原本还困扰着少女的怨灵瞬间全部哀号着消失殆尽。

“灵梦！”

少女惊喜地抬起头看向灵梦，只见玄爷正载着灵梦向这边飞了过来，而他们身后，则是留下了双眼变成蚊香状挂在树上昏过去的雾雨魔梨沙。

“切…”

看到灵梦已经打败她的徒弟脱身到这边来了，魅魔轻啐了一声，然后淡定地站了起来。

魅魔的腰间和左肢已经完好如初，只留下了之前被冈格尼尔贯穿时在衣服上留下的裂痕，很明显是在用怨灵缠住少女的时候用了治疗的魔法：

“灵梦，刚才打了那么久，你们恐怕也累了，不如咱们今天不打了，下次再说吧。”

听到魅魔的话，灵梦脸上的表情犹豫了一下，似乎是在考虑要不要答应。

然而听到魅魔的话，少女皱了皱眉头，打开了魅魔和灵梦的资料。

【魅魔的状态

HP：985200/1325200

MP：14500/6320000】

【博丽灵梦的状态

HP：868000/968000

MP：1216000/3250000】

（靠，魅魔这家伙在坑人！）

哭笑不得的少女凑在灵梦的耳边轻声说道：

“别答应她，她的身体虽然已经恢复了，但是体内已经没有什么可用的魔力之类的了。”

原本还有些踌躇的灵梦在听到少女的话后愣了愣，然后才露出一副生气的样子看向魅魔：

“好啊魅魔！你居然想骗我！你根本就没什么力量了吧！快乖乖让我封印！”

发现自己的谎言被戳穿，魅魔明显地愣了愣，然后把愤怒的眼神投向了躲在灵梦身后的少女。

不管怎么看都太明显了，灵梦在听完少女的悄悄话后突然知道了魅魔在说谎，一看就是少女告诉的。

“可恶的小丫头，你给我记住！”

留下了一句狠话，魅魔在灵梦反应过来之前就拍打着翅膀窜到了空中，然后伸手抄起正悬挂在树枝上雾雨魔梨沙，准备飞走。

“想逃？没那么容易！”

看到魅魔准备逃走，灵梦连忙操纵着几张灵符射了过去。

“哼哼，不陪你们玩了。”

魅魔轻轻笑了笑，然后一转身就凭空地消失在了原地，一点痕迹也没留下。

“诶！？怎么消失了！？”

灵梦瞪大了眼睛看着因为失去目标而掉下来的灵符，满脸惊讶。

少女也是惊诧万分，原本以为魅魔又隐身了的她刚才打开了地图，却发现并没有看到魅魔和魔梨沙的光点，这种情况只有两种可能——一是魅魔的技能已经能逃脱系统的勘察，二是那两个人真的已经不在这个地方了。

郁闷地关上地图，少女立刻被扑上来的灵梦抱住了。

“红你没事吧！？刚才你跟那个魅魔打架我居然没注意到，玄爷也没跟我说，没受伤吧！？”

一边说着，一边还准备动手脱少女的衣服。

“没事没事…”

少女尴尬地推开灵梦，如果不是灵梦的眼神里的确是只有关心的情绪，少女恐怕会以为扑上来的是个女色狼，“只是被怨灵穿过之后有点虚弱而已…”

“应该是被吸了点生气，休息会儿就好了。”

玄爷轻轻地说道。

“嗯嗯，以后打架这种事情我来就行了，红你就在一边给我加油就好。”

灵梦坐在玄爷背上点了点头，一脸深以为然的表情。

少女有些无奈，不过很快她又注意到了一件事。

（明明已经打败了那么多敌人，为什么没有一点经验值？）

少女带着疑惑的表情看了看一路上打过来的灵梦，却发现在灵梦的名字下面，那个“邀请组队”的按钮居然还是亮着的。

（原来我忘了和灵梦组队啊啊啊啊啊！！）
