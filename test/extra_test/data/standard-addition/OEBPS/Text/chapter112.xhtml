第二卷50——压倒性的差距

莉莉丝视角

眼前的精灵村落陷入了一片火海之中。

透过不知何时被解除的结界，随处可见那些被遗弃在地上的，破损的武器和防具。

这大概是因为它们的所有者，此刻已纷纷化作『肉块』散落在附近的缘故吧。

那些家伙的死状可谓相当凄惨，有被刀刃切断的，有被狂风撕裂的，有被冰柱贯穿的，有被巨岩捣碎的。

但对于没能马上咽气的倒霉蛋来说，不用承受多余的痛苦便上路的他们，反而是幸运的吧。

拜此所赐，惨叫，悲鸣，哀嚎，恸哭不绝于耳。

“还好之前见识过真正的『地狱』……不然现在认错了也不奇怪啊。”

『是不是做得太过了？』。

感叹之余，我再次询问起自己的内心。

『那么，你打算等到被抢走后才悔恨吗？』

结果，一个熟悉又陌生的声音用相同的方式反问。

“……怎么可能……”

主人大人是个十分温柔的人，这一点我比谁都要清楚，因为自己正是被那份温柔给拯救了。

但是，我同样清楚，主人大人并不会吝啬自己的温柔，更不会说出只对我一个人温柔的那种话。

能让人联想到火焰般的温暖光芒，无论我多么严严实实的抱在怀中，总归会有些许透过缝隙挥洒出去。

而那必定会吸引到其他的，渴望感受这份『温柔』的家伙们的接近。

“如果变成那样就麻烦大了。”

要是展开所谓的『主人大人争夺战』的话，没有属于自己魅力的我，恐怕胜算并不高。

毕竟去除『莉莉丝』的要素后，我也只是个随处可见的普通人。

所以，就算是不择手段，也要在『害虫』靠近之前将其歼灭。

“绝对不会让给任何一个人的……主人大人的温柔只属于我……”

那份温柔将我从绝望的深渊中拉扯了出来，还让我明白了原以为不忍直视的世界，是如此的多姿多彩。

哪怕它就跟寓言里提到的那根垂落到地狱中的蜘蛛丝一样，不与其他人分享一起攀爬便会断掉，我也一样会将它占为己有的！

“唉，话虽如此……可这次大概不会只是挨骂那么简单了吧……”

很轻易的就能想象出，主人大人大发雷霆，甚至把我赶出家门的未来。

一丁点都不想和主人大人分开……但是，主人大人被抢走的可能性，我更加不能容忍其存在。

“……请……给……我……”

正当我的内心斗争激烈进行中的时候，不远处传来了失魂落魄的声音。

我扭头一看，发现在焚尽精灵村落的火焰对面，有一名步履蹒跚的男性正朝着我缓缓靠近。

“啊，搞定啦。”

那是曾经被冠以『勇者』名号的家伙。

“……奖……励……”

“真快呢。”

对，曾经。

因为他现在的模样，无论如何都不会与『勇者』相称。

“事情都办完了吗？”

原本光鲜亮丽的铠甲早已被血液和污渍悉数侵染，巨剑的剑刃也因为不断碰撞变得坑坑洼洼。

而他身上散发出的，死气沉沉的氛围，更是会让所有人敬而远之。

“有把她们好好杀掉吗？”

“……和……我……做……吧……”

“果然没法对话了，真可悲。”

由我这个将他变成如此模样的罪魁祸首来说似乎有点不恰当，但除了『可悲』外我也找不到合适的形容词。

被『指向性魅惑』击中后就是会变成这样，此时此刻他的脑海中仅剩下『推倒我』这一个念头。

不管是友情，亲情，哪怕是爱情之类的，全都会被抛到九霄云外。

“所以我才一点都不喜欢……”

正因为这个技能不会差别对待任何人，所以只要我一个不小心，就极有可能伤到主人大人，然后好不容易构筑起来的与主人大人的羁绊估计也会毁于一旦。

为此，我曾经提议过将双眼给弄瞎，虽然看不见主人大人帅气的身影很可惜，但怎么都比发生意外来得好。

结果那天我被臭骂了一顿，说什么要爱惜自己的身体啦，没必要担心啦，相信我不会乱用啦之类的主人大人，完全变成了『滥好人』模式。

不过这样也只会让我更加喜欢主人大人啦～～～诶嘿嘿嘿嘿～～～

“……和……我……做……”

“……你居然还在……话说我好不容易想起一些开心的事能不能别打扰我。”

“……交……配……”

“够了够了，别烦我了，你爱咋地咋地去。”

好不容易进入一下妄想的世界就被拽出来了好烦啊，但是放着这个失去自主意识的提线木偶不管，他肯定会不厌其烦的纠缠我。

无奈之下我只得打了个响指，将『指向性魅惑』给解除了。

伴随着我眼中的红光渐渐消失，勇者的双眼也开始取回原来的清澈。

“……”

在短暂的几秒沉默后，他突然抱着脑袋撞向了地板，双手狂乱的抓挠起来。

“啊啊啊啊啊啊啊啊啊——！！！”“呜呕呕呕呕呕呕呕呕——！！！”

一边呐喊一边呕吐的家伙我还是第一次见。

“呃，抱歉，太久不用我好像忘记技能效果了……”

看见这种反应我才想起来，『指向性魅惑』并不包括事后抹去记忆的功效，也就是说他仍然记得自己将整个精灵村落屠杀殆尽的这件事。

被迫回顾双手沾满同伴鲜血的全过程可真惨。

“你！！！这！！！家！！！伙！！！”

惊人的杀意，哪怕没有洛斯的『敌意感知』我也能理解到眼前这名勇者，全身上下的每个细胞都在叫嚣着『杀了我』。

我不认为这种情况下还能通过对话来解决问题，也没有那种打算。

拳头定胜负可以说是『异世界』的通用法则。

“杀了你！！！绝对！！！绝对要杀了你！！！”

“啊，真可怕……呃，不对，好像也也没那么可怕啦……”

我早就不是过去那个被人稍微威胁一下就会吓得发抖求饶的家伙了，来自败犬的乱吠不会对我造成任何影响。

“说到底动手的也不是我啊，明明是你自己把她们一个一个亲手杀掉的。不管求饶的还是逃跑的，都没放过啊？”

“你这！！！你这！！！”

“硬要说让计划产生偏差的，也是身为『勇者』的你的精神『过于强韧』的缘故吧，虽然对我来说某种程度上属于『愉快的误算』啦。”

本来打算让他中招后冲上来试图推倒我，接着我用镰刀将之反杀，最后再亲自消灭剩下那些绝望的精灵们的。

然而不知道什么原因，『指向性魅惑』最初的几秒没能发挥全部的功效，那家伙居然居然保留了一丁丁点的理智。

还好死不死刚好把我随口说的那句『只要你杀光了这个村落里的精灵我就给你奖励哦～』给听了进去。

结果显而易见，他高效率的代劳了，给我省了不少事。

“可恶可恶可恶！！！去死啊啊啊——咳咳咳！！！”

“……差不多到极限了吗……”

见到『勇者』的口中吐出的大量鲜血，我无奈的叹了口气。

我从头到尾欣赏完那场『杀戮秀』，所以很清楚眼前的家伙早就不行了。

村落里的精灵在发现『勇者』异变后，可没有乖乖等死，而是拼命的抵抗了。

然后被『指向性魅惑』控制，只按照最初的『杀光』指令行动的勇者，对于那些精灵们的攻击既没有『防御』也没有『回避』。

强的一方实力受限，而弱的一方底力全开，因此产生了我最乐于见到的结果——也就是两败具伤。

“身体都变得破破烂烂了居然还能活到现在，只能说不愧是『勇者』了吗？”

“不饶你！绝对绝对！绝对绝对不饶你！！！嘎啊啊啊啊啊！！！”

“……”

不知为何，与对我恨之入骨的那家伙相反，我并不怎么恨他。

但是，我知道这种只属于胜利者的『余裕』会因为立场的扭转而发生剧变。

假如倒在地上的人是我，肯定也会做出与『勇者』相似的举动吧。

所以，就这么放对手一条生路的蠢话我肯定不会说。

“结果能做的也就只剩下给你个痛快了呢，有什么遗言吗？”

“去死去死去死去死去死去死！！！”

无视了那家伙最后的诅咒，我将召唤出的镰刀高举过头顶，然后径直挥落。

他的身体就这样被贯穿，抽搐了几下后，便再也不动弹了。

“……”

远没有预想中那样的满足感，反倒是觉得越发空虚了起来。

“他明明机会那么多，为什么非要堂堂正正的战斗呢？”

不把『断空斩』瞄准我，不用『空间转移』把我弄走，甚至不干扰我发动『指向性魅惑』……

唯一想到的可能性，就是他至今为止从未苦战过吧。

“算了，想太多也没用。赢了固然好，但也没必要鞭尸。”

拍了拍脸颊让自己冷静下来后，我朝着和洛斯约定好的汇合地点开始前进。
