第四卷10——可以预料的结果

“……不会坐视不理？就你这小豆丁居然还敢口出狂言？”

我是不会反对那位『好心大叔』继续数落人的，但问题是也要注意下别一个劲的去踩玲菲的雷行不？

“豆，豆丁……！？”

她的耳朵和尾巴都已经若隐若现了哦，说明火冒三丈到了难以继续控制魔力去维持『伪装』了呢……

唉，这么下去两位铁定会打起来，而且要真打起来我真是站哪边都不合适。

站『好心大叔』那边等于背叛玲菲，肯定不行……她刚刚挺身而出保护我，替我出气的行为都让我挺感动的呢。

但如果站她那边，两个人一起上去痛殴那个『好心大叔』的话……把他揍得像其他『找茬事件』里的小混混一样灰溜溜滚蛋倒是简单，可之后要怎么摆脱『大姐头』这一称呼可就难咯。

所以怎么办呢？除非他们不打起来，否则我肯定怎么着都不是……啊！

对哦，既然他们打起来就麻烦大了，那不让他们打起来不就万事大吉了？

真佩服自己能在这种时候『灵光一闪』啊，虽然想到的方案有些尴尬，但现在也管不了那么多啦。

于是，我硬着头皮闯到了那两个人之间，并强行挤出了笑容。

“两，两位住手……Lov，LOVE ＆ PEACE~争吵是不对的哟！”

坏消息是，在主人大人以外的家伙面前摆剪刀手果然比想象中的还要更加丢人，下次打死都不会这么做了……

至于阿黑颜？那才不是给你们看的东西啦！

““？？？””

而好消息嘛，就是效果比想象中的还要好，不止玲菲，就连那个『好心大叔』也愣住了。

不愧是我，『天才魅魔』可不是浪得虚名哦~

然而就在我满以为大功告成而放松警惕的时候。

“你他妈的是在糊弄谁啊！？”

那位『好心大叔』也不知道发了什么疯，突然青筋暴起，把还能自由活动的左手直接挥了过来。

“唔哦！？”

还好体内那只『超巨型史莱姆』反应够快，自动张开了『史莱姆防壁』替我防住了这一下，否则我绝对要挨巴掌的。

然而『防住』并不代表没事，我因为受到惊吓而后退了——接着脚一滑，一个踉跄便摔坐在了地上。

“亚玄知亲！”

肯定是以为我被打伤了吧，玲菲立刻跑到了我的身边，想要扶我起来。

“汝还好吧！？”

“呃，还好还好，问题不大……”

不想让她担心，所以我没像平时那样喊疼。

可实际上，屁股是真开花了，一点都不好受啊！

讨厌的大叔！居然随便动手打女孩子！一点都不『好心』！这必须得教训一下才行了！

但正当我思考着该如何报复他的时候，周围那些看热闹的冒险者们窸窸窣窣的议论声传了过来。

“那家伙在干什么啊？”

“为什么突然就动手打人？”

“而且打得还是个那么小的小女孩！”

咦，总感觉他们的眼神里没有之前那种『尊敬』和『仰慕』的感觉了呢，取而代之的是『担心』与『同情』……

如果是这样的话，可能那些人之后就不会再喊我『大姐头』了……但为什么他们的态度会发生转变？

稍加思索后，我明白了其中的缘由。

那群人是受了洛斯的影（洗）响（脑）才把我认为是个『狠角色』的，但实际上并不知道我的『真实水平』。

而刚刚的那一巴掌，直接粉碎了他们先前的『印象』，『谣言』自然也就不攻自破咯~

好极了！虽然跟计划得不太一样，但结果就是我想要的——啊不对不对，其实一切都是我算好的哦，我是故意去挑衅那位大叔让他打的呢~

考虑到事情已经办妥，我兴冲冲的打算告诉玲菲现在万事大吉了——却被她的模样吓了一跳。

“汝·这·厮——”

她的狐耳和狐尾都已经完全显现，怎么看都是气炸了的状态。

说起来，这只白毛萝莉好像从一开始就不知道我的打算来着……现在更是因为我『被打』而火冒三丈了。

“我？我怎么了？老子可是巨斧的刚特伽尔，别以为你是兽人族我就会怕你！就算那只黑猫再来我也不会怕了！”

她溢满而出的杀气让站在后头的我都冷汗直流，但哪怕面对这样的玲菲，那个找茬的家伙也鼓起了勇气没有退缩。

真是无谓的自尊啊……我本来还想着因为他一巴掌『阴差阳错』的达成了我的目的，现在我可以不追究他的责任的……甚至如果玲菲要追我还可以帮着拽住一下……但为什么好死不死非要往雷区里头闯？

以为自己能赢得过那只白毛萝莉吗？这已经不叫『勇敢』而叫『鲁莽』了。

当然，我的抱怨没能传到他那里，只见那位找茬大叔不知道从哪掏出了一把应该就是名号由来的，几乎和的身高差不多持平的长度的超大斧头，用左手扛到了肩上。

“呵呵，呵呵呵~呵呵呵呵呵……”

然而极具魄力的拔武器动作，换来的只是玲菲的冷笑。

那笑声是挺瘆人的，所以看见那家伙紧张到摆正了架势，我并不惊讶。

而玲菲则是挥了挥手。

“还想着抵抗干嘛，汝已经没有赢的可能了呐~”

像是不想做无用功似的这么对他说。

“……哈？”

“汝已经输了呐……所以现在妾身可以再给汝一次机会。立刻低头跪下，对妾身和亚玄知亲认错，这样妾身便会绕汝一条狗命呐。”

这种说法有够狂妄，但不知为何我就是觉得那只白毛萝莉能做得到这种程度。

“……你这家伙！看不起谁啊！在那说什么屁话！？”

那位找茬的大叔因为被彻底的小瞧了而同样生气了起来，他用脚猛的往前一踏，一个箭步便冲到了玲菲跟前。

老实说，用那种『大块头』等级的身材还能做出如此敏捷的动作，他说不定是个有点水平的冒险者呢……

但就在所有人都觉得白毛萝莉要变成『白色肉块』之前，玲菲叹了口气。

“真是不见棺材不掉泪呐，『跪下吧』。”

接着说了这么一句话。

对，她什么都没做，从头到尾都站在那一动不动，只是说了一句话而已。

“咣当——！”

接着下个瞬间，巨斧就从那位找茬的大叔手中滑落，砸到了地面上，而他本人也狠狠的摔了一交。

“什……么！？这，这是……怎么回事！？”

“……所以妾身都说了，汝毫无胜算的呐……”

“可，可恶！为什么站不起来！？你这混蛋到底干了什么！？”

“妾身可没干什么呐……说不定只是汝的身体已经认识到自己并不是妾身的对手，所以直接就臣服了呐~”

不不不，怎么看都是玲菲你用什么招式控制了那位找茬大叔的身体吧，但感觉好像不是之前用过的『狐媚』，是什么新技能吗？

“唔——！”

“看在汝已经五体投地的份上，妾身就给汝最后一次机会呐~赶紧对着妾身和亚玄知分别说上一句『对不起，我错了，下次再也不敢了，求求你们大人有大量饶了我吧』，妾身便会放过汝呐~”

这么说完后，玲菲干脆的丢下了那个找茬大叔，跑到我的身边抓住了我的手。

“如何？妾身是不是很厉害呐~替汝报仇了呐~”

“啊，这个……”

其实报不报仇都可以啦——这种话肯定是说不出口的，不知道该聊些啥的我最终决定询问一下她刚刚用的技能的事。

我是抱着玲菲愿不愿意回答都可以的心态问的，没想到她很爽快的就点头了。

“妾身用的是『言灵』呐……是母上前不久传授给妾身的呐……虽然那人很讨厌但实力还是得认可的呐……”

原来她到现在还在生妈妈的气啊，我还以为早就该忘了呢。

“原来如此……但这种事告诉我没关系吗？在『武斗祭』上我们可能会碰上的诶？”

“没关系呐，虽然要知晓『真名』才能使用，但只要知晓了就没有抵抗的手段了呐~所以告诉亚玄知亲也没关系呐~”

“这样哦……”

没想到在这边的世界里也存在着在地球的漫画书中才存在的技能，总之我还是先别告诉玲菲，自己叫莉莉丝得了……

“反正，那个叫什么刚伽……的家伙已经构不成威胁了呐，现在咱们差不多该办正事了呐~”

对哦，我们来这里是为了换零钱的来着，然而——

“你们……两个！居然还在那……有说有笑的！甚至无视我！等我起来，绝对不会放过你们的！”

大哥，你在哪乖乖趴着不好吗？玲菲很明显都懒得理你了，不去惹她等个一阵子她不就放过你了？

果不其然，火气又被挑起来的白毛萝莉这次直接下达了『砸穿地板』的命令，导致那天到我们换完零钱决定离开为止，那位找茬的傻瓜的惨叫声一直回荡在冒险者公会中。
