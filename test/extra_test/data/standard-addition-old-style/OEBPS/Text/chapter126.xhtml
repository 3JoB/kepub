第三卷02——入学考试·笔试部分

“……真没想到这玩意在这种地方还能派上用场……”

入学考试试卷的『姓名』一栏上，我填的是『亚玄知』这几个字。

再怎么说也不会直接写『莉莉丝』啦，那么做可是会引发骚乱的。

……讨厌，你们干嘛露出『这家伙居然在考虑那些』的眼神嘛？

这种程度的事我当然会考虑到啊，为了不给主人大人添麻烦！

而，而且我绝对不会因为想听主人大人那句『就算与世界为敌我也要保护你』就故意暴露身份的哦！

倒不如说在发色和瞳色通过主人大人的魔法暂时变成了银白色和粉红色的现在，我和之前的模样已经有了很大区别，就算说自己是『莉莉丝』也不一定会有人信的，很安全的~

嗯？你们问那个名字是不是在哪里出现过？

这都不记得了？第二卷的开头啊，冒险者公会会长马尔特为了能让我参加『领主选举』，特意给我准备了这个『伪名』，也可以说是『伪造身份』呢。

顺便一提洛斯当时的『伪造身份』是『亚玄知』的妹妹『亚玄可』。

啊……！说到这个我才想起来，前不久我和洛斯才接到那个矮人大叔的通知，说我们已经在选举中获胜，成为了『哈利塔克斯的领主』——名义上的。

然后按之前谈好的，马尔特现在掌握了实权，我们只需要作为这座城镇的『看板娘』偶尔露露面就行了。

既能获得额外的报酬，还能名正言顺的居住在『领主宅邸』，何乐而不为呢？

“如果那份『背景资料』别那么蛋疼就更好了……！”

我也很清楚一介平民假如想当领主，反对的声音会此起彼伏，但要是有『贵族血统』各个方面就会简单很多……但为什么得和那个纵欲过度而死的肥猪前领主扯上关系啊！？

“就算是『伪造身份』，我和洛斯居然都是他的女儿什么的也太扯了……”

之所以一直没在公众面前露面，是由于母亲想从父亲的咸猪手中把我们保护起来，而让我们逃跑了——这样圆的谎。

槽点是不是多到大家都不知道从哪里开始吐比较好了？

当时肯定有极力反对，只可惜洛斯却“从小就跟姐姐大人相依为命？好棒！”的投了赞成票，使我输给了数量上的暴力，只能接受了。

“喂！那边的！你在碎碎念什么？”

“呃，抱，抱歉……”

差点忘了现在还在考试中呢，不能随意讲话。

“不要想着作弊哦，监控很严格的！”

“……”

尽管监考的老师这么说啦，但我认为会在笔试部分作弊的人应该一个也没有。

『人类的主要活动区域在卡纳大陆的？（请填方位词）』——送分题，南方。

『罗德里克王国的首都是？』——能来这里的还有谁不知道？隆德兰嘛。

『位于卡纳大陆东边的大片森林区域被人类称为？』——兽精之森。

『请写出F级冒险者能承接的委托（一种）』——去餐馆打杂。

毕竟你们看，题目就这种难度，考的全是『最低限度的常识』，只要不是傻子基本都能回答出来。

『只不过数量有点多啊……』

总计100分的考卷，每道题都值1分，也就是说总计有100题，好烦！

更见鬼的是，哪怕笔试部分的100分你拿满，换算成入学考试的最终分数也只有仅仅30分。

嗯，没错，这所学院的入学考试分数占比并不是笔试5成，实技5成，而是笔试3成，实技7成这样的。

我差点就起了干脆现在睡觉，等下实技拿个满分完事的念头了。

『不不不，轻敌可不行，要认真……』

不过让你们看我答卷也没啥意思，趁这个机会我来介绍一下准备要就读的『骑士魔法学院』好了。

首先从上面提到的笔试与实技分数占比来看，大家应该很容易就能发现这所学院的教育方针是以『实践』为主『理论』为辅的。

因为这座『骑士与魔法学院』，本质上属于一个『冒险者培训机构』。

院方自然不会希望在此就读的学生，在毕业后踏入职场（战场）时无法发挥出应有的水平便活活送命的。

所以这所学院，用主人大人的话说，属于入学容易毕业难的那种，他并不推荐那些只是『慕名而来』的学生到这浪费时间和金钱。

不过从刚刚候考区挤满了人来看，还是有相当多的人打算考这里呢。

果然『顺利通过毕业考核时，将授予B级冒险者资格证』这点十分有吸引力啊。

假如不清楚B级大概是个什么程度，可以看看备注的说明部分。

那么介绍完教学方针后，接下来的应该就是课程分类了吧。

『骑士学院』那边，基本上就是教会学生如何使用各式各样的武器来进行战斗的地方。

有『剑术科』，『枪术科』，『盾术科』，『骑术科』……等一大堆我虽然感兴趣，但很清楚自己没那种本事去碰的科目

毕竟以我的力量值，拿着毫无重量的『死神镰刀』都没法空挥一百次就会气喘吁吁，根本不可能拿起真刀真枪呢。

想必大家也发现了，『骑士学院』并不是专门培育那种拿着长枪大盾的骑士的地方，但这个名字仍旧是在产生了激烈的讨论后被确定下来了，只能说果然『骑士』这个职业都被大家所憧憬吧……

而主人大人让我考的『魔法学院』，传授的课程就跟学院名完全一致了

以『魔法类』为主，面向的是将来希望以『魔法』作为主要攻击与支援手段的冒险者们。

有像是以攻击力见长的『元素魔法科』，以使役契约魔物为核心的『召唤魔法科』，制作魔导具的话必修的『符文魔法科』，造成大量增益减益的『支援魔法科』等等种类繁多的科目。

别说我偏心，介绍『魔法学院』的时候更详细，毕竟我压根就没打算去管『骑士学院』的事啊，知道一点传闻都不错了。

真想知道那边的情况，就去问洛斯吧，她在另一个考场考试来着。

“还有最后5分钟，大家记得检查一下不要有错填漏填的题目。”

听见监考老师这么说之后，我抬起头看了看周围的其他考生，发现他们似乎都是一脸轻松的模样。

看来并不只有我一个人觉得试卷很简单呢，笔试部分大家应该都是以满分为目标的吧，所以分不出优劣也很正常。

但接下来的『实技部分』可完全不是这么回事咯。

就让我这个天才魅魔莉莉丝教会你们这些人，什么叫做『实力上的差距』吧~
