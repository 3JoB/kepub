第一卷33——好感度似乎上升了1点

莉莉丝视角

“好痛哦————！不对，好像也不是那么疼啦。”

恢复意识后我猛的从地上坐了起来，然后看着周围被自己坠落砸出的小坑默默的叹了口气。

就跟预想的一样，坠落的过程中因为脑袋先着地所以受到了『致命伤』，身体反倒在一瞬间就恢复原状了。

“倒是省得我在地上滚来滚去了……”

“盯——！”

啊，现在看起来并不是应该对自己逐渐掌握这项能力而庆幸的时候，毕竟刚刚掉下来的并不止我一人。

“怎么样，你还好吧？”

顺着视线望过去，如果我的眼睛没被摔出问题的话，那洛斯应该没有大碍，毕竟她的手脚看起来完好无损，脖子也没扭到奇怪的方向去。

真不愧是猫科兽人，十分擅长减轻坠落伤害呢。

“x……”

“嗯，你说啥呢？”

“谢，谢谢！”

本来一会儿摆向左，一会儿摆向右的尾巴，最终还是垂了下去。

“……哦吼～～～我刚刚有点耳鸣没听清，你可以再说一次吗？”

“不可以！你在撒谎！刚刚你已经听见了！”

“切～”

立如芍药坐如牡丹，不开口就招人喜欢……如果她能保持着刚刚准备道谢前的样子的话，还是挺挺可爱的啦。

“啊，但你为什么要谢我啊？”

“不是在骗人？……是真的不知道？谢谢你是因为刚刚掉下来的时候你把我推到上面去了，所以我才没受伤。”

“哦，好吧。”

『原型』的发动条件是受到致命伤，如果我拿洛斯当垫子自己摔个断手断脚也太不划算了，所以我才决定自己砸向地面比较好……

当然这种事还是别告诉她了，我还没有蠢到故意降低别人对我的评价的程度。

“你看起来，很古怪……而且，为什么没事？刚刚不是摔成肉饼了吗？”

“……原来我摔成肉饼啦……”

看来以后还是少做点这样危险的举动好了，万一哪次『原型』发动失败我不就凉了？

“我发现了，你可能……不是坏人？虽然行为很像个坏人，说的话也可能很像个坏人，但实际上应该……也许还是个坏人？”

“喂！喂喂喂？”

在最后不是应该反转一下，说一句『实际上应该很善良』吗？我可是清纯又可爱的派系的哦？

“爹地说过，被人帮助了就要道谢才行，所以我才谢谢你的。”

“嘛，其实我也不能说毫无责任……毕竟一开始带着你飞上天的是我。”

“……说的也是，那我收回自己的道谢。”

“咳咳咳！？啥？什么鬼，刚刚那是社交辞令啊！你还真的说收回去就收回去啊！？”

“噗——”

出乎意料的，我的吐槽却换来了洛斯的嘲笑——不，准确的说这还是我第一次看见她笑出来。

“虽然你看起来像是坏人，但确实相当奇怪呢？”

“那啥，你这两句话有哪一句是在夸我了吗？”

是我理解能力太差了还是怎么地？这两句话之间有转折的意思吗？

“知道我的能力后，还愿意这样和我聊天的，你是除了爹地外的第一个。”

“你是指『辨识真伪』？”

“……差不多吧？村里的其他人不管是见到爹地，还是见到我，都会躲得远远地。”

“这我还真不知道。”

臭老头，你女儿不爱和人打交道的原因，可能不止是你光顾着锻炼她了哦，看起来能力遗传也是问题之一。

“你真的不害怕吗？”

“害怕啥？”

“爹地说大家都很喜欢说谎的，也害怕自己的谎言被人揭穿。”

“嗯，那倒是没错。”

据说人平均说三句话就有一句假的呢。

“那你为什么不害怕？”

“呃，所以我要害怕什么？”

“这种能力啊！能分辨别人有没有在说谎的能力！”

“这有什么好害怕的，我还能从被摔成肉饼的状态活过来呢？”

我觉得光论哪个更不讲道理些，『原型』要比什么『辨识真伪』强无数倍吧，顺便一提『指向性魅惑』我到现在一次都没用过就是了。

真要让我害怕的话，那也得主人大人说出像是……唔，光是想想就觉得太可怕了所以不敢想了。

“唔，但是这不一样吧？类型不同？”

“你这家伙真的是一点都不懂得变通哦。我们干脆来做个试验好了。”

“shiyan？”

“啊，就是玩个游戏啦，猜拳你会不会？”

石头剪子布，我觉得应该不管那个世界都通用的游戏之一。

“会，但是为什么？”

“因为我要用事实告诉你，那种能力一点都不可怕。”

“咦？”

“总之，我想出『布』。”

“嗯？你没在说谎……但是哪里有把自己的想法提前告诉别人的？”

“来嘛，石头剪子布～”

就和我预想的一样，洛斯出的是剪刀，而我出的是石头，我赢了。

“为什么！？”

“还要再来吗？”

“再，再来！再来一次！”

“那我还是想出『布』。”

“这次是在骗我对不对！我看得出来的！”

“嘛，石头剪子布～”

结果洛斯出的是石头，而我出的是布，结果同上，我赢。

“为什么！？为什么你还能赢我！？”

“因为我有用心理暗示啦。。”

“心理暗示？”

“就是把『石头』当做『布』，『布』当做『剪刀』，『剪刀』当做『石头』给自己施加了这样一条想法来着。”

“为什么要这样？”

“当然是为了骗过你啦。”

不得不说这么做需要花点心思，结果导致我现在有点累了，主要是心累。

“唔，听……听不懂！”

“那我们换种方法，你先想好要出什么？”

“……想好了……”

洛斯瞪大了双眼盯着我看呢，看起来还是在使用她的能力的样子。

“你想出石头。”

“……什么？我……”

“哎呀，先别告诉我你想出什么，我刚刚看起来在撒谎吗？”

“没有……？”

“那，你不想出石头。”

“这也没有撒谎……咦？为什么？”

“看吧？所以我说是有弊端的。”

“为，为什么？你应该至少有一句话是在撒谎吧？”

“还不懂哦？刚刚的两句话，连我自己都不知道是不是在『撒谎』，你怎么可能分辨的出来呢？”

说白了，洛斯和她老爸的能力，一句话来概括就是『人形自走测谎仪』，判断的不是事物的『真伪』，而是说话的人有没有出现说谎时会有的『精神波动』而已。

在地球上的话，对付测谎仪可以用药物，强烈的暗示，包括高浓度巧克力什么的，导致这种机器的成功率甚至不到7成哦。

“唔……也就是说，一定要对方认为自己在撒谎我才看得出来？”

“是啊，现在我再说一次，『你想出石头』。”

“！你，你在撒谎！”

“没错。”

通过洛斯之前的反应来判断，她应该想的是『布』或者『剪刀』，不然反应不会是『……什么？我……』而是『为什么知道？』这样的惊讶。

这么推理之后我再说『你想出石头』就自然而然会被洛斯当成谎言来判断了。

“唔，我……我从来没想过这些，爹地也没教过我……看不出来，你的脑子那么灵光……”

“看不出来是什么鬼，我可是天才魅魔莉莉丝诶？”

“这句话是在撒谎！我能看出来！”

“……你这家伙真的讨人嫌……”

好吧好吧，其实我也很清楚自己没那么聪明，只不过是从以前看过的一部漫画里拿点东西来现学现卖罢了。*1

实际上要应付这种能力还是相当麻烦的，至少我没有厉害到能时时刻刻都保持这种和她勾心斗角的状态，光是动动脑子就很累了。

“你觉得，我讨人嫌？”

“呃，这个嘛……说话委婉点，少用点那种能力的话，应该不至于？”

不管怎么说，现在我应该和这只猫耳幼女的关系拉近了一丢丢了吧？算不算完成她老爸给的『和洛斯做朋友』的任务了？

不管算不算，之后我都要向那个臭老头要只猫耳幼女蹭个爽。

“就算不用能力，我也能感觉出来你散发出了邪恶的气息！”

“啧！我就不明白了，不过就是想摸点毛茸茸的东西哪里邪恶了啊？”

“毛茸茸？”

“就是指像你这样头上有猫耳，还有猫尾的女孩子啦，最好年纪再小一点可以让我抱在怀里的！可以蹭来蹭去的！”

“……为什么要做这种事？”

“为了舒缓压力，懂吗？平日我积累的压力太大了。”

谁叫主人大人如此的木头呢，稍微理解一下我的难处该多好。

“看起来，不是在撒谎的样子……”

“对你撒谎我有钱赚吗？没有，那我撒谎做啥？”

就算有钱赚也要看看能赚多少啊对不对，如果能赚到主人大人主动向我示爱等级的『钱』的话，那倒是不错哦。

“你果然很奇怪，虽然不一定邪恶，但是很奇怪。”

“……算了，到这种地步我也不指望你能换个词来形容了……”

大概就和她不希望我叫她『洛斯酱』一样吧，看来我也放弃比较好。

在回黑猫村落的路上，我们两人都在聊着这些毫无营养的话题，但当时的我完全没想到事情会进展的如此迅速。
