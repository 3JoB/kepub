第一卷44——契约魔法

莉莉丝视角

“虽然那个完全不理解我的可爱之处的臭老头说这座迷宫是他的祖先和以前来到这边的异世界人合力建造出来的啦……”

一边举着火把，我一边沿着那个矮人大叔给的地图上画好的路线前进。

“但老实说这明显看起来就像是年久失修的危险建筑的内部啊。”

周围斑驳的墙壁，随处可见的蜘蛛网，甚至还有被搜刮一空的宝箱，就算是傻子都能看得出来这里蛮长一段时间没人管理了。

“算了，跟我关系不大。”

毕竟我又不是来欣赏风景的，还有更重要的事情得去完成。

说着我就从背包里装着的一大摞纸中取出了其中一张，然后向着上面画着的魔法阵注入了自己的魔力，至于注入魔力的方法就和以前使用『魔杖』的时候一模一样呢。

很快，被充能的魔法阵就发出了淡蓝色的光芒，确认它的质量过关后，我就把整张纸揉成一团，随手丢到了地上。

“反正这里都这么脏乱差了，乱扔垃圾也没关系吧～开玩笑的～”

事先声明，这可不是在乱扔垃圾哦，而是行使『契约魔法』的一个重要步骤。

用那个臭老头的话来讲，远古时期的『契约魔法』并没有多少用处，其中一个问题就是使用者只能和『比自己弱很多的对象』进行契约。

而另一个问题，也是主要的……就是大部分人认为，比起让『契约对象』冲上去打架，还不如自己『欧拉欧拉』敌人来得更加帅气一些。

“没毛病啊……如果可以的话我也想『欧拉』敌人！”

要不是这幅身板那么脆，攻击力还是防御力都低得要死，我肯定会像其他主角那样疯狂装逼的诶！

啊，跑题了，现在主要还先讲讲『契约魔法』——几年前，事情出现了转机。

有一个来到这边的『异世界人』提出了一种『契约魔法』的新的使用方式，成功解决了『只能和比自己弱很多的对象契约』的这个问题，才免于让『契约魔法』彻底遗忘在历史长河之中。

当然还是不帅，没错，帅的问题是解决不了的，暂时。

他的方法就是先和大量的极其弱小的魔物进行『契约』，然后再让那些较为强大的魔物将他的『召唤物』吃掉——这样重复个几轮之后，就连强大的魔物也会抵挡不住数不清的『契约魔法』的约束力，最终成为新的『契约对象』。

“如果我没记错的话，这是某款游戏里发生的事情吧……但会把它带到这个世界来用就算是『异世界人』的本事啦～”*1

说起来我也是同款的『异世界人』诶，不知道以后有没有那些『经验』能用得到的地方呢？

总之，我现在就在进行『第一步』，也就是和大量的弱小的魔物进行『契约』，而方法就像之前提到的那样，把这些注入了自己魔力的『契约魔法』纸张丢到这座迷宫的各个角落去。

“一张两张三张四张……哇，这真的是个超级大工程……”

最初的时候我还在想着，能不能干脆直接点就让那些厉害的魔物把我做好的『契约魔法纸』一口气全吃下去……但马尔特会长却告诉我，稍微有点智商的魔物都不可能吃这种带有不明魔力的东西……结果就是我还是只能按照老方法一步步来。

“可话又说回来了，我感觉自己已经踏过了大半个迷宫了诶，背包里的纸都少了一大堆了，怎么到现在连一个魔物都没看见啊？”

这是迷宫里哦，就算是看起来像『废弃』的迷宫但好歹还是个迷宫啊。

骷髅兵？蝙蝠？哥布林？宝箱怪？亡灵？反正我全都没看见，什么都没有，空荡荡到有点吓人了。

像主人大人带我去过的城外草原区走两步就能碰见一只波波兔呢，没准还能碰见一大堆波波兔……

“我也没带那种驱虫香之类的东西啊，这么不科学的吗？”

按照『经验』来判断，出现类似的情况，那么接下来一定会发生一些比较奇怪的事，于是我决定提前开启『魔化』做好十二分的准备。

“………………”

然而不管是我小心翼翼的前进，还是大胆的向前冲，绕了迷宫整整一圈都看见出口了还是什么都没发生。

对，什么都没发生，这个迷宫里甚至连一只魔物都没有哦，整个迷宫真的像是被彻底荒废了那样。

“诶，不是吧？马尔特大叔你给我推荐的这什么鬼地方？”

说好了来这里和低级魔物契约的，但这也得有低级魔物才行啊，空荡荡的一座迷宫我跟谁去契约哦。

“这条迷宫只剩一条岔路，就通向BOSS房间的那条道我没走过了，我肯定不可能直接去打BOSS的呀……”

只要BOSS比波波兔强上一丢丢，我都有自信会被干得很惨，各种方面的可能会被『干』得很惨……当我傻吗直接去？

“算了算了，那就走到门口再退回来吧，开个全图。”

这么想着，我又重新向着迷宫的深处前进。

“唔，这一天乱丢垃圾的次数可能比我上辈子还多……”

因为是第二圈，所以沿路都能看见之前自己丢在地上的纸团，数量应该是完全没变过，所以应该来说这个迷宫里是真的没什么魔物了。

至少，不存在那些会吃我丢的纸团的低级魔物。

就在这时，我在岔路的尽头处看见一个牌子，上面写着『直行，BOSS房间。右转，恢复之泉』。

“啊哈哈哈哈哈……这真不愧是有『异世界人』参与设计的迷宫啊，『大战之前必有补给』是吧？唉……”

有的时候真的感觉这个世界真的挺欢乐的，当然还是因为来这边的『异世界人』有不少的关系。

“嗯？”

正当我这么想的时候，却注意到放置了『恢复之泉』的房间里正向外散发出蓝色的光芒。

好奇心驱使我走进了那个房间，不对应该说是走『近』了那个房间，因为我没能『进去』。

“咦？”

有着什么蓝色半透明的薄膜把我给拦住了，不是屏障，因为一点都不硬，更像是肥皂泡泡那样。

“诶～”

用手戳一下就会陷进去，用力戳的话就会陷进去的更深，收手的话就会弹回来，也不会被弄湿。

“这是什么鬼？”

还在疑惑中，眼前的那个『薄膜』就开始发生变化了，它不断向内收缩，越缩越多，然后渐渐不再死死塞住房门口的部分。

“……？”

叫我进去吗？不会有危险吧？不过马尔特会长说过他会通知主人大人，就算出了什么事也会有人来救我的对不对？

一想到主人大人就好像注射了强心剂那般，使我的胆子变大了不少，因此我便鼓起勇气走进了类似『恢复点』的房间内部。

“噢噢噢……喔噢……”

眼前的景象使得我惊呆了，原来之前那个像『薄膜』一般的东西实际上并不是一层『膜』，而是『什么东西』的外壁。

理论上来讲，眼前的这玩意应该是一只超级无敌巨型的大史莱姆，但是由于它实在太大，以至于我觉得用『什么东西』来指代可能更加合适。

如果一般意义上来说，正常的史莱姆大约就一只迷你犬那么大的话，大一定顶多也就是藏獒的程度吧。

但是这家伙几乎把整个放有『恢复之泉』的房间完全的占据住了，都已经能碰到天花板了哦。

这似乎还是缩小了一圈后才变成这样的，如果完全放开来没准能比得上主人大人的家。

“等，等下，等等？”

仔细一看我才注意到，它的整个身体呈现半透明的状态，而且是非常清澈的那种，里面根本看不见一丝杂质，就连刚刚在房间外看见的光似乎也是它的身体放出来的。

“该，该不会……？”

有种说法是史莱姆可以吸收外来的物质，改变自己的种类……那么眼前的这家伙吸收最多的东西，怎么想都是这口『恢复之泉』了。

“…………”

“！？”

是注意到我在盯着它看了吗，这只超巨型史莱姆挪动着自己的身躯往一边靠了靠，然后让出了一小条缝，刚好足够我直接走到『恢复之泉』旁边。

不会错的，它早就不是那种普通的史莱姆了，绝对是已经进化到了具有相当程度智能的超级史莱姆了诶！
