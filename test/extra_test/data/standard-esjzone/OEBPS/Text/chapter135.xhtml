第三卷11——朋友

莉莉丝视角

“虽然姐姐大人可能认为没必要，但我还是觉得和舍友搞好关系会更好哦。”

在闲聊中度过的午休快结束时，洛斯对我说道。

“这就是你开学才几天就收获到一大堆跟班的经验之谈吗？”

“姆，姐姐大人别再抓着之前话不放啦……我现在可是很认真的在为姐姐大人出谋划策的说。”

“出谋划策？”

“我觉得这是达成姐姐大人目的的最快捷方式哦～”

尽管猫耳幼女还打算继续解释下去的样子，但她的那群『跟班们』（？）没给这么多时间——在午休结束的铃声响起时就把这位将来的『会长』给半拖半拽的带走了。

然后一转眼来到了傍晚，在当天的课程很普通结束后，我回到了自己的宿舍，站到了房门前。

“洛斯没说我也是知道的啦，确实是这么个道理没错。”

跟舍友混熟的话，请她们在主人大人美言几句也不那么难了吧——比如像是『莉莉丝各方面表现得都很不错』之类的。

这样主人大人应该就会认为我已经有很大进步，允许我离开学院回到家里，重新过上之前那种没羞没躁的生活啦～

是个值得一试的计划呢。

“但执行起来好像没那么简单……”

混熟什么的，讲得直白点至少也是要成为『朋友』的等级吧。

虽说也算是打过照面了，所以事到如今我不至于在她们的面前表现得太困窘——但想要『畅谈』感觉还是有点难。

而且前世的自己也不是那种擅长交际的类型，所以没什么经验……

“唉，要是能简单的做到，可能主人大人也不会把我送到这学院了。”

想到这里，我『啪啪』的拍了拍自己的脸颊。

“我可是天才魅魔的莉莉丝酱！才不会因为这点困难就——”

边给自己鼓劲边推开房门，然后又猛地将其合上了。

这，这才不是退缩哦，我这么做是因为——

『锵——！』

物体碰撞的声音从门的另一侧传出，证明了刚刚自己并非眼花，而是真的有什么东西朝着这边飞了过来。

慢一步脑袋就要被打烂的那种。

“……什么鬼？”

安全起见，再度推开房门时我已经把『史莱姆屏障』提前张开了，然后看见的情况是：

飘荡在空中的青蓝大火球。

散发着光芒的银白十字架。

凝聚着旋风的翠绿魔法阵。

“对不起打扰了。”

偶尔也是会有的啦，这种走错房间的情况。

“等下！亚，亚玄知，汝是叫亚玄知对吧！”

突如其来的搭话让我停下了脚步，回头一看妖狐一族自称是『公主』的那位正慌慌张张的瞪着我。

没记错的话是叫『玲菲』来着？

“汝来得正好呐，快帮妾身一把！”

“……哈啊？”

先不提咱就是一个不想蹚浑水的吃瓜路人，有用这种语气找人帮忙的吗？

虽然心里这么想，但我更在意到底发生了什么事。

仔细一看我的另外两位舍友，爱薇儿和蕾蒂娅——她们正用着明显不能说是『友善』的目光盯着那只妖狐公主。

稍加思索，我得出了结论。

“你们有仇报仇有怨报怨撒～”

“喂！汝何出此言呐！？”

“谁叫你之前要用『狐媚』搞她们，现在被发火不是理所当然的？”

换做我的话肯定会气到当场回敬一番呢，真是难为那两个人忍到现在咯。

“才不是因为那件事好吗！妾身可是有好好取得原谅呐！”

“……那为什么？”

“妾身也不知道呐……不过是想起来今天该打理尾巴了，所以回到宿舍后便拿出梳子梳顺毛发而已！”

玲菲边说着边抖动了一下身后的那三团毛绒绒的玩意……似乎确实是因为疏于打理而失去了应有的造型。

“总之妾身这几天肯定没再做过招她们记恨的事！但她刚刚却突然的两眼发光的扑过来了！”

“……这边这位是这么说的哦，你们两有什么要反驳或者解释的吗？”

事到如今好像也错过了脱身的时机，所以在听完玲菲的话后，我转身面向了爱薇儿和蕾蒂娅。

“……哈……哈……嗨……诶……哈……”

回答我的人是爱薇儿……呃好吧，这到底算不算得上回答得打个问号。

“摸……摸……摸……摸……”

“什，什么……？”

“摸！让我摸一下！超级想摸一下！那毛绒绒的尾巴啦！居，居然能见到那么可爱的生物！好想摸！超级想摸！想把她抱在怀里拼命的摸啊啊啊啊啊！”

“……”

我转身就走。

“喂！汝这是在干嘛？临阵逃脱吗？”

“放弃吧，那不是你能战胜得了的对手。”

站在那里的大概已经不是名为『爱薇儿』的女孩而是别的什么东西了。

那副喘着粗气，眼睛瞪大到充血，身子不停抖动的模样，虽然可能会很失礼但叫做『怪物』应该不过分……

“那汝也不该就这样丢下妾身啊！”

“我跟你非亲非故的好吗！为啥要替你消灾解厄啊！？”

“咕奴！若，若汝能帮妾身从这个危机里逃脱，那让妾身收汝为下仆也未尝不可啦……”

“滚！你当我傻的啊！？”

瞧瞧这是人说的话吗？呃好吧这家伙原本就是妖狐啦。

无论如何被她这么一说后我产生了干脆把她交给爱薇儿的想法——啊对哦，为什么不呢？

“诶，诶诶诶！？汝，汝可不要有什么危险的打算呐！妾身一族的尾巴可是魔力的源泉，要是被弄坏了会出大麻烦的！”

“省省吧，哪会有什么大麻烦？不就是身体变得虚弱然后任人蹂躏吗？”

“……呜哇好可怕，汝是怎么知道这么详细的后果的？”

“……诶？我就是随便那么一猜哦？”

因为以前看过的小薄本里不是常有类似的桥段吗？就是那种『尾巴都是敏感点』之类的。

顺便一提我『魔化』后长出的尾巴是可以当做『小老弟』来用的哦，不过欺负洛斯时自己是站在主动的一方，并没有让什么人拼命撸过就是……

“总，总之不要再把妾身往那边推了！妾身可不想落入那家伙的手里！”

“那你就拿出点诚意来啊，诚意！”

“咕呜呜——那，那就做朋友如何？妾，妾身已经做了最大限度的让步了，能和身为妖狐一族公主的妾身做朋友可是相当荣耀的事哦！”

尽管很想吐槽一句为什么她会认为我愿意和这种到现在还一幅高高在上态度的家伙成为朋友，但就为了最终目的而言她的提案挺诱人的。

虽然有趁火打劫的嫌疑，不过这应该是个好机会吧？

“拜，拜托啦！之，之后妾身的尾巴，给，给汝摸摸也不是不行……！”

“……咦？明明对爱薇儿那么抵触？”

“对那种狂战士会抵触不是理所当然的吗！？”

说的也是呢，她的表现让我这局外人都觉得可怕了……话说真的没问题吗？我是指角色崩坏方面。

“咦对了，蕾蒂娅呢？该不会这样的怪物其实有两个吧？”

“你才怪物呢！别把我和她混为一谈好吗！？”

“诶诶……”

我随口一句的吐槽居然被听见了吗，尴尬。

不过仔细一看蕾蒂娅确实边拽着爱薇儿的衣服边张开着结界，所以那个已经『怪物化』的家伙才没能冲到这边，干扰我和玲菲的谈话。

“别的不说，玲菲你现在欠我个很大的人情哦！”

“妾，妾身总觉得自己一下子陷入了严重的负债泥潭……”

正当妖狐一族的公主陷入烦恼之际，一直处于狂乱状态的爱薇儿突然像是恢复理智的对蕾蒂娅说道。

“姆，也差不多该放手了吧？”

“呃？”

“还『呃』什么啦，蕾蒂娅你难道想要代替玲菲让我抱抱吗？”

“唔——”

“虽然不能把头埋进那毛绒绒的尾巴大口的闻味道，也不能把它们抱在怀里蹭来蹭去很令人遗憾，但仔细想想之前品尝过的蕾蒂娅的身体不也有着相当程度的等级吗？”

说出来你们可能不信，但蕾蒂娅真的快被吓哭了。

“放心啦，今天还有别的舍友在，我会收敛一点不影响她们的睡眠的～”

“我的睡眠会被糟蹋掉的啊！！！”

大概是敌不过爱薇儿的胁迫吧，修女小姐很干脆的松开了拽住她衣服的手，并且将光的屏障解除了。

没记错的话早在刚开学的时候她好像就被爱薇儿『无惨』过呢……

“诶嘿嘿，这样阻挡在面前的就只有亚玄知小姐了～”

“……”

“其实刚刚我有注意到哦，亚玄知小姐对玲菲所说的『朋友』一词起了反应呢～是这样吗？”

“……”

“那样的话和我也成为『朋友』如何？只要把那个趾高气昂的毛绒绒生物交出来就行啦～”

“……我觉得，还是应该分先来后到的，这里背叛她的话各种意义上都显得不太好诶……”

我摇了摇头拒绝了爱薇儿的劝诱，身后的那位妖狐公主，你可以多感谢我一番哦，最好之后拿出像样的诚意。

“唔姆，现在的话还能附赠一位可爱的修女哦？”

“诶嘿嘿，汝放弃吧～妾身看人的眼光可是一流的，妾身的友人才不会因为一丁点的追加条件就——”

“啊，您请慢用。”

趁着玲菲还在那慷慨陈词的时候，我绕到她身后轻轻一推，把她送进了爱薇儿的怀抱中。

“……诶？”

“啊啦～这可真是～？”

“没有永远的朋友，只有永远的利益……虽然我想这么说，但爱薇儿小姐还请手下留情，别把玲菲玩得太……啊……”

沉浸在毛绒绒触感里的那位似乎已经听不进去我说的话了，不过这样一来『交朋友』的目的也算是达成了？实在是可喜可贺可喜可贺呢～

“亚玄知你个大叛徒噫呀～～～～～”——玲菲

“这，这还真是意料之外的天国降临！呀哈！”——爱薇儿
