各自的死斗（2）

拓宽的战场仅因为三个魔物的登场就一口气关窄了。

把我方的尸体当作路走的异形向前迈步──仅仅如此士兵就战栗地让开了道路。

一时兴起砍了一刀人就飞了。

只有鲜血染红了草原的绿。

“接下来好像是主戏了呐──莎夏，盖伦，阿尔──”

“好的”“噢”“明白了”

接到克丽丝塔的指示，盖伦和阿尔跑了出去。

首先是阿尔向悠然等待的食人鬼（Maneater）乌诺投掷了短剑。

“武技（Arts）──飞翔刃（Fly Knife）”

从左右同时迫向乌诺的短剑闪过以肉眼追不上的高速剑闪。

应该被收在鞘里的两柄剑不知什么时候完成了拔剑。

啪嗒一声，失去力量的短剑落在了地上。

“哈，很好──武技（Arts），爆进突！”

一直线加速的盖伦的剑带着光，在挥剑后的空档朝着乌诺的腹部迫近。

和猛烈前进的盖伦不同，乌诺的动作可以说非常缓慢。

但是，那绝不是他的行动晚了。

只是动作没有浪费，倒不如说那个动作快得令人难以置信。

“好慢啊”

盖伦的剑尖快要碰到躯干的瞬间，乌诺抽身一转避开了剑刃。就那样利用回转的力量，用右手拿的剑像要砍倒盖伦一样挥动剑刃。

“咕⋯⋯好重⋯⋯”

盖伦勉勉强强用剑腹接住，但被压倒性的魔物的力量弹飞了，一边被吹飞一边在地面上打滚。

《敬爱的信徒祈祷──给弱小者治愈的加护──神圣治愈（Heal）》

《罪人落于冰狱之地。断绝、隐没、消失、赎罪吧──魔法技（Magic Arts），冰剑乱舞》

和莎夏的治愈飞向盖伦同时。

漂浮在半空中的七把剑带着意思袭击乌诺。

“吼，比刚才愉快不少──”

刹那，乌诺的眼睛睁开了。

浑浊的眼睛确实地捕捉到了从空中迫近的七把剑。

两柄剑挥动了四次──上下的二闪描绘了如光一般的残渣被挥到底后，二闪加速像画弧一样重叠。

初击破坏了两把，回避了三把，次击破坏了两把。

“武技，弧月四闪吗，人类的技能也相当有用──！”

但是就连那个对克丽丝塔来说也不过是诱饵而已。

一瞬间逼近的克丽丝塔推出两把细剑。从背后是刚才回避的冰剑迫近。

这个夹击的形式正是克丽丝塔期望的理想展开。

“斑火！咕──！”

那是以肉眼追不上的高速突刺。

七把剑灵活得让人觉得在玩一样。

虽然乌诺用两柄剑巧妙处理了，但速度是克丽丝塔压倒性地占上风。

“真快呐──但是，火力略微不足啊──”

乌诺采取的手端仅是以力量压制。

妥善活用肉体能力的差距，一边吃下突刺一边走到前面并挥剑。

对乌诺来说只是微弱的伤害。另一方面，他有剑刃哪怕只命中一发这个奢华的女人也完了的确信。

“区区怪物少瞧不起人──冰结之花绽开──魔法技（Magic Arts），冰花剑凛”

银色的细剑缠上了冰。

在挖去肉的前方冰的蔷薇带刺地开放了。

“嘎啊啊啊啊啊啊啊啊啊啊啊啊啊啊！”

“A级冒险者，冰帝，克丽丝塔・尼洁・布兰里赫特。这是杀你的人的名字。你最好作为带去冥土的土产记住”

克丽丝塔看向在战场旁伫立的少女，松了口气。

看来自己好像在这里尽全力就足够了。

◇

“哈哇哇哇哇⋯⋯⋯⋯⋯⋯”

“咕噜噜噜噜⋯⋯⋯⋯⋯⋯”

艾夏因为格里芬压倒性的巨躯害怕得发抖了，另一方面格里芬也在害怕艾夏发出的强大魔力。

不过对艾夏来说，眼前的格里芬看起来就像是在猎物前狞猛地笑着。

“那，那个─，克丽丝塔小姐，危急，超危急，我──！”

可是，克丽丝塔只是看了艾夏一眼，马上就回到了自己的战斗。

“怎，怎么这样──”

当然，看到她的激烈战斗的话也明白余裕不多，但没想到会被抛弃。

这难道是背叛。

是为了独占娜哈特大人的膝枕的战略不是吗。

当然和艾夏不同，克丽丝塔认识到了格里芬正在害怕的事实，因为觉得不需要援军所以才采取了那个行动。

可是对无从知晓那个的艾夏来说，对克丽丝塔感到了不稳定的气氛，魔力越发进出了，不过本人完全没有那个自觉。

“咕噜噜⋯⋯⋯⋯⋯⋯⋯⋯”

像是同情越发变小的格里芬一样，水的大精灵告诉艾夏道。

“在害怕哦，那孩子──怎么办呢？”

“呋欸，那个，诶，说话了，啊嘞？”

那是全身用水构成的美女。和小人们不同，样子相当煽情。虽然身体是水，但是赤身露体。艾夏困惑地看了精灵。

“当然会说话啦，因为别看我这样，我也是水的大精灵──温蒂妮的分裂体──被好吃的魔力吸引了，（๑ゝڡ◕๑）。请多指教呢，精灵之子哟”

“那个，这个，我才是，请多多指教”

艾夏尽管不知所措，还是回应了借给自己力量的精灵。

“好的，被拜托了。那么，那个要怎么办呢？”

“欸都，如果可以不战斗就解决的话那是最好的──啊，但是，为了娜哈特大人的膝盖要有牺牲⋯⋯哈！我在想多么残忍的事⋯⋯”

艾夏刚一露出狞猛的笑容，就马上摇头了。

“真是个忙碌的孩子呢，青春啊─”

精灵带着好像老太婆似的笑容说了那种话。但是她的表情一瞬间不高兴地歪斜了。

“那么，那边的人类怎么办？”

精灵机敏地对人的感情作出反应。

精灵有着看清善恶的直觉。被作为代价的魔力根据精灵不同会激烈变化，必然的精灵魔法会选择使用者。正因为艾夏有着半耳长族（半森精灵）的魔力所以才叫出了她。

那样的她也看了人的魔力后，作出了判断。

至少不想让那些变成自己的契约者，温蒂妮那样直觉。

“呋欸？”

听到的是策马疾驰的声音。那个身影毫无疑问是我方的骑士。

但是──

“艾夏小姐，增援来了──”

那样说的同时，魔道骑士团（Magic Knights）的团员从马上挥下了挥起的剑。

“增援？真是相当，嘛，搞笑──”

注意到时在艾夏不知道的地方，小小的战斗结束了。

三个头被水刃斩飞。

“──哎呀，抱歉契约者哟，傻瓜早早地退场了”

结果娜哈特的准备也以多余的关照告终了。

艾夏只靠自己的力量就摆脱了所有处境。不过，仅仅只出现了三人的理由是因为格兰迪亚斩杀了大半。

“哈哇哇，到底怎么了──”

艾夏困惑地嘟囔。

◇

“嘎⋯⋯怎么可能⋯⋯⋯⋯！”

一个魔族倒下。尼古尔多冷淡地俯视着那个。

“怎么，找错人了吗”

沾满血和肉的战斧只留下了割断地面的痕迹。

“好像是呢，公会长──那么就由我来让它断气──”

《──炎啊，从天上出现在地上的业火啊，把我眼前的敌人烧成基石。炎魔法（Flame Magic），送葬业火（Funeral Flare）》

气息奄奄的鬼（奥加）特雷斯一被业火包围，就像存在没了一样化成灰消失了。

“干嘛，伊莉娜。抢人头可不好哦？”

“如果您也瞄准了娜哈特小姐的膝盖的话，我觉得现在马上自首被砍掉头就好”

“这是冤罪！而且，立即死刑什么的！让老夫辩解下啊”

“驳回，变态必须死”

一边和伊莉娜对话，尼古尔多一边环视战场，看向中央和左翼。

哪边都大概势均力敌吧。

“老夫中央，伊莉娜你左翼。去支援”

伊莉娜想要对那样的尼古尔多的话点头，但没能做到。

异变不只是伊莉娜，战场全体都在一瞬间把握了。

因为无论怎么说，浮在空中的那个是连隐藏自己的身姿大概都做不到的巨躯。

红色的翼只是拍打就产生了风。引起的风就像锐利的刃一样。

塞满了狞猛的嘴的白银的牙，那些看起来像是稍微隐藏在深处。

同时，那个嘴角染上了深红。

“火竜──怎么可能──”

这是在开玩笑，谁都这样想。

谁都进行了现实逃避。

到底谁能瞧不起那个呢。

嘴角产生的业火看起来就像尼古尔多过去见到的火山的喷火口一样，不能跟人使用的魔法一样看待。

啊啊，但是说起来，感觉就在最近见过相同的魔法。

落在交易都市的闪电。

正和那个一样吧。

不过，攻击范围和被制御的娜哈特的魔法没法比吧。

“──！公会长，后面──”

伊莉娜很冷静。

看见危机后马上展开了防御魔法。

可是，对那个有多大意义呢。

一条光穿过时，战场沦为了地狱。
